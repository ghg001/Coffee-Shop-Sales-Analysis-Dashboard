# Coffee-Shop-Sales-Analysis-Dashboard

### Project Overview
This project analyzes sales performance data from multiple coffee shop locations Astoria, Hell’s Kitchen, and Lower Manhattan  to uncover insights into revenue trends, top-performing products, and customer purchasing behavior. Using Excel’s Pivot Tables, Charts, and Slicers, the interactive dashboard provides a clear view of monthly, weekly, and category-level sales performance.

### Objectives
- Identify sales trends over time (month-over-month revenue growth).
- Analyze daily and hourly transaction patterns.
- Determine top-performing product categories and items.
- Compare location-based performance across branches.


### Tool used
- Microsoft Excel: Data cleaning, transformation, and dashboard creation
- Pivot Tables & Charts: Aggregation and visualization of key metrics
- Slicers & Filters:Dynamic interactivity for multi-location comparison

### Key Insights
- Revenue grew steadily from $81K in January to $166K in June, showing consistent upward performance.
- Friday recorded the highest sales volume across all days of the week.
- The Coffee category dominated sales with 58,416 transactions, followed by Tea (45,449) and Bakery (22,796).
-Barista Espresso generated the highest total revenue ($91,406), followed by Brewed Chai Tea ($77,081) and Hot Chocolate ($72,416).
